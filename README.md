# SEZ AI 2026

Welcome to SEZ AI! This project is a full-stack React application powered by Vite, Express, and the Gemini API.

## Hosting on Replit

This project is fully configured to be hosted on Replit.

1. **Import the project** into Replit.
2. **Set Environment Variables**: In the Replit Secrets tool, add your `GEMINI_API_KEY`.
3. **Run**: Click the "Run" button. Replit will automatically install dependencies and start the development server using the `.replit` configuration.

## Tech Stack
- Frontend: React 19, Tailwind CSS, Framer Motion, Lucide Icons
- Backend: Express, Vite Middleware
- AI: Google Gemini API

## Development
To run locally outside of Replit:
```bash
npm install
npm run dev
```
